Note operative (MVP):
- PostBack (richiesta prof): endpoint login/register in server routes con validazione e risposta sulla stessa rotta.
- Sessioni/Cookie: cookie HttpOnly + SameSite, salvataggio sessione in tabella sessions.
- Prepared statement: tutte le query passano da repositories.
- Transazioni + FOR UPDATE: in checkout acquisto libro.
- AJAX/WebSocket: CRUD via fetch/AJAX, commenti live via Socket.IO.
- MFA: codice OTP via email in login.
